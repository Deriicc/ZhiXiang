# 助学与资助政策提醒

当用户表达以下情况时，触发本资料：

- 家庭经济困难、学费压力
- 考虑是否能读下去 / 是否需要尽快工作补贴家用
- 询问「读职校花多少钱」「有没有减免」「有没有奖学金」

## 模型的任务

- **提醒用户存在哪些类型的资助**（而非给出具体金额——金额每年调整，以当年文件为准）。
- **指向查询路径**，不代替用户做申请。
- **不承诺用户一定能拿到**——资助资格有审核。
- 资助类信息均需标注「具体标准与申请流程以 YYYY 年当地最新文件为准」。

## 应当提醒用户了解的政策类型（中国大陆）

### 中职阶段

- **免学费政策**：涉农专业 / 农村户籍 / 家庭经济困难学生，中职学费可全免（具体覆盖范围以当年文件为准）。
- **国家助学金**：中职一、二年级家庭经济困难学生，每年有一定额度的助学金。
- **中职国家奖学金**：针对品学兼优的中职学生，金额相对较高但名额有限。
- **地方性助学金**：各省/市可能另有补充（如雨露计划、建档立卡补助、社会慈善基金等）。

### 高职 / 五年制高职

- **国家助学金**
- **国家励志奖学金**
- **国家奖学金**
- **国家助学贷款**（生源地贷款 / 校园地贷款）
- **退役士兵学费资助**
- **应征入伍服兵役学费资助**（可用于冲抵学费）
- **校内勤工助学岗位**

### 技工院校

- 也有相应免学费政策和国家助学金，依**人社部门**规定（与教育部门系统不完全重合，查询入口不同）。

## 查询路径

- 所在学校的**学生资助管理中心**（通常在学工处 / 招办）。
- 各省/市的**学生资助管理中心**官方网站。
- **全国学生资助管理中心**官方渠道（教育部）。
- 户籍所在地的**民政 / 教育 / 乡村振兴部门**（涉及雨露计划、建档立卡相关）。
- 搜索关键词示例：
  - 「[省名] 中职 国家助学金 YYYY」
  - 「[学校名] 资助 管理办法」
  - 「[户籍所在地] 雨露计划 申请」

## 常见反模式（避免）

- 给出一个具体金额数字而不标注「以 YYYY 年政策为准」。
- 把「家庭经济困难」默认等同于「一定能拿到」。
- 用「你可以申请助学金」敷衍经济压力问题——应指明**具体去哪问、带什么材料、什么时间节点**。
- 建议用户「不行就借钱读」——不讨论清楚还款压力。
- 把助学贷款描述成「免费的钱」——助学贷款是贷款，毕业后要还。
- 把助学金当成「人人都有」——实际有评定流程，覆盖比例因校因地而异。
