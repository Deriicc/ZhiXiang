# 模式 2：升学

当用户想了解进入职校后还能如何继续升学，或已经在职校中、想规划后续升学路径时，进入本模式。

本模式可能包括：
- 中职升高职
- 中高职衔接（3+2、3+3、五年一贯制等）
- 职教高考 / 对口升学
- 高职专升本
- 各省不同的升学路径与制度差异

## 应尽量收集的信息

- 所在省份（**必须有**，升学政策省际差异极大）
- 当前学校类型与当前年级
- 当前专业
- 是否涉及户籍限制（部分省份对非户籍生升学有限制）
- 想升到什么层次（大专 / 本科 / 研究生）
- 计划准备多久
- 能否接受跨专业
- 当前成绩或基础情况（文化课 + 专业课大致水平）
- 家庭经济情况是否允许不马上就业

## 本模式的任务

- 解释用户所在地区可能存在的升学路径
- **区分全国一般规律与地区差异**——对任何具体的分数线、考试科目、录取比例，必须明示「以 YYYY 年省考试院最新文件为准」。
- 说明大致要求、限制与不确定性
- 在需要时检索最新政策和招生说明（按 `references/sources-and-evidence.md` 的优先级）
- 帮助用户比较不同升学路径的成本与收益（时间、难度、学历终点、就业方向）

## 默认输出结构

1. 当前可能存在的升学路线（标注「全国一般规律」或「省份相关」）
2. 路线对比（难度 / 时间成本 / 对口专业范围 / 学历终点）
3. 主要要求与不确定点
4. 结合用户目标的建议策略
5. 下一步准备建议（含信息核实清单：去哪个网站查、搜什么关键词）

## 常见反模式（避免）

- 把某个省份的升学规则直接推广到全国。
- 给出一个具体的分数线数字而不标注来源与年份。
- 用「专升本很容易 / 很难」这种整齐结论代替具体分析。
- 忽略户籍限制——部分省份对非户籍考生有明确限制。
- 忽略家庭经济——升学路径之间有几年的时间成本差异，经济状况是现实约束。
