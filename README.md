## 这是什么？为什么做这个？

在今天的教育叙事里，职校生常常不是被真正理解，而是被提前归类。很多人还没来得及认识一个具体的人，就已经先接受了一套现成判断: 分数低一点，就意味着能力差一点；去了职校，就像是“掉队”了；没走普高这条路，好像人生的上升通道也跟着一起变窄了。围绕职业教育的许多讨论，表面上是在谈学校、谈专业、谈出路，实际上谈的却是等级、体面和谁更值得被看得起。

这种偏见最残酷的地方，不是它说得难听，而是它常常被说得理所当然。它会出现在亲戚饭桌上，也会出现在老师的劝说里，出现在同龄人的玩笑里，出现在那些看似“为你好”的建议里。很多职校生不是在某一个瞬间被打击，而是在一次次被比较、被矮化、被替代发言的过程中，慢慢学会怀疑自己。好像只要进了职业教育，就必须先为自己的存在辩解，再去谈梦想、兴趣、能力和未来。

可真正活在这条路上的人，面对的从来不是一句“读职校也有前途”就能解决的事。有人在中考后被仓促分流，甚至还没来得及理解不同学校、不同专业、不同升学路径之间的差别，就被要求马上做决定；有人明明想继续往上读，却连本省职教高考、对口升学、专升本到底怎么走都找不到清楚、可信、讲人话的信息；有人在顶岗实习里遇到超时劳动、工资被扣、协议不透明，却被默认为“学生嘛，忍一忍”；还有人最难受的根本不是学业，而是每天都要承受“你们这种学校”“你以后也就这样了”这种话，一点一点把自尊磨薄。

更现实的问题是，社会并不真的在认真听职校生说话。关于他们的内容很多，但真正从他们的处境出发的内容很少。官方文件讲制度，却不回答“我现在该怎么办”；媒体报道喜欢讲逆袭故事，却不讲大多数人真实要付出的代价；社交平台上的讨论不是猎奇，就是羞辱，要么把职校生当笑话，要么把他们包装成励志样板。很多 AI 也没有好到哪里去: 它们会用空泛的安慰掩盖现实的难题，会在你最需要一个清楚判断的时候，递给你一套看起来完整、其实并不贴身的标准答案。

`职想`就是在这种背景下做出来的。它不是想替谁做人生决定，也不是想替职业教育“洗白”。它更不想顺着那套默认的等级眼光，再把职校生解释一遍、安慰一遍、规训一遍。它想做的事情更直接一些: 当一个职校生、一个正在分流边缘徘徊的初中生、一个焦虑又犹豫的家长，真的带着问题来问时，能有一个工具不回避现实，不重复偏见，不用空话糊弄人，而是把信息讲清，把路径摊开，把代价说明，把那些混在一起的事实、评价、偏见和羞耻一点点分开。

所以你可以把它理解成一个专门为职校生写的 AI 对话 Skill。但如果说得更准确一点，它其实是在试着回答一个更根本的问题: 当一群人长期被忽视、被误解、被低估时，能不能有一个工具，至少先把他们当成值得被认真对待的人。

## 它能干什么

职想目前能支持六种对话场景，根据你说的第一句话自动判断你在哪种情况里：

| 场景 | 典型问题 |
|---|---|
| 🏫 **择校** | 选什么类型的学校、什么专业、如何比较不同学校 |
| 📚 **升学规划** | 职校后怎么继续升学——中高职衔接、职教高考、专升本等 |
| 🏆 **竞赛** | 职校生可以参加哪些竞赛、值不值得参加、怎么选 |
| ✏️ **学业求助** | 课程、作业、考试、某个知识点的学习困难 |
| 💪 **偏见与社会压力** | 因职校身份被羞辱、轻视、否定，或自我怀疑、内化羞耻 |
| 💼 **就业与实习** | 顶岗实习、校企合作、第一份对口工作、劳动权益 |

## 安装 / 使用方法

### 在WorkBuddy中使用

### 在 Cursor 中使用

1. 将 `zhixiang` 文件夹放入你的 Cursor Workspace 的 `.cursor/skills/` 目录下：
   ```
   your-project/
   └── .cursor/
       └── skills/
           └── zhixiang/      ← 把整个文件夹放这里
               ├── SKILL.md
               ├── modes/
               └── references/
   ```

2. 在对话中直接提问即可，不需要任何额外指令。

### 在 Claude Code 中使用

将 `zhixiang` 文件夹放入：

```bash
~/.claude/skills/zhixiang/
```

然后在对话中直接用就行。

### 启用小红书搜索（可选）

小红书搜索需要本地运行 `xiaohongshu-mcp` 服务：

1. 克隆并启动服务：[github.com/xpzouying/xiaohongshu-mcp](https://github.com/xpzouying/xiaohongshu-mcp)
2. 在对话里说「**去小红书查一下**」或「**小红书上怎么说**」即可触发

没有配置这个服务也完全可以正常使用，只是拿不到小红书独家内容。

## 示例对话

```
​User: 我知道他们有偏见，可我还是会忍不住觉得是不是我真的低人一等。

ZhiXiang.skill:
你说你"知道他们有偏见，但还是会忍不住觉得低人一等"——这很诚实。能意识到偏见存在，本身说明你并不真的认同那些话。但问题是：有些话听多了，就算你知道它不对，也还是会像伤口一样隐隐作痛。

这不是你的软弱，是人之常情。

先区分两件事：

他们说的"低人一等"，指的是什么标准？

——是成绩？是中考分数？是坐在哪个教室？

这些是具体的事实：你确实选了职业教育的路径，分数线确实有高有低，这是客观现实，没什么好否认的。

但"低人一等"是一个价值判断——它默认了一套标准：成绩好 = 高等，成绩差 = 低等。

问题是：这个标准是谁定的？它对所有人适用吗？

能考高分的人，会修机器吗？能上普高的人，一定会做饭吗？一个城市之所以能运转，不只是靠坐在办公室里的人，还需要焊电路的、开车的、做饭的、护理病人的。这些能力，值多少钱？

他们的评价体系里，从来不问这个问题。

你会"忍不住觉得"，是因为从小到大，你听到的所有信号——学校、家长、社会——都在告诉你：只有一条路是"正确"的。

你被这套逻辑养大的，现在它已经内化了。就算理智上知道偏见存在，感情上还是会痛。

但感觉不等于事实。

如果有一天有人再说"读职校就是低人一等"，你可以试着在心里问自己一句：

"你说的'一等'，是用什么量的？"

这句话不用说出来，只是留给自己的一把小小的刀，划开那些大词的空洞。

```

```
User: 我现在最怕的是进了一个管理很差、实习很坑的学校，怎么提前看出来？

ZhiXiang.skill:
这个问题问得特别好，也特别重要——很多学生和家长都是进了学校之后才发现踩坑的，而那时候退出的代价已经很大了。提前识别"管理差、实习坑"的学校，其实有规律可循。

一、管理差的学校，通常有这些信号
招生层面的预警

"包就业""包分配"——越是把就业说得太肯定的学校，越值得怀疑。真正就业好的学校不需要用这个词招生。
招生老师特别积极催你报名、反复催促"名额不多了"，制造紧迫感让你没时间好好了解。
不愿意告诉你完整收费清单，或收费项目特别多、额外费用频繁出现。
招生宣传和实际差距很大——官网照片特别高大上，但实地去看发现不是那么回事。
学校运营层面的预警

打听一下老师的流动性——频繁换老师的学校，通常内部管理有问题。
学生退学率高不高？可以侧面问问："你们学校读满三年的人多吗？"
学校有没有学生投诉的渠道？有没有学生会、意见反馈机制？如果什么都没有，学生就是单方面被管理。
学校频繁被媒体报道（尤其是负面报道），在天眼查或裁判文书网上有纠纷记录，这些都可以查。
教学层面的预警

专业实训设备很老旧、甚至没有实训场地，学了三年都是理论课。
课程表和培养方案含糊其辞，问起来就是"到时候安排"。
对职业技能证书（1+X证书、技能等级证书）不重视，或者考出来没什么用。
二、实习很坑的信号
实习是最容易出问题的地方，因为学校可能在这里面有利益。

需要警惕的情况

实习和所学专业完全不对口：比如学的数控，却被送去电子厂做流水线。这很可能说明学校没有正经的企业合作资源。
实习没有报酬或报酬极低（低于当地最低工资标准的60%），且学校说"这是规定"——这类情况在部分地区确实存在，但要判断学校有没有克扣或抽成。
学校强制指定实习单位，不让学生自己找。这种操作背后可能有"人头费"。
实习期间学校不闻不问，出了问题找不到老师。
打听一下往届学生的实习经历——"上一届去哪实习了？累不累？能不能学到东西？"这个问题很直接，能问出真实情况。
一个重要判断标准：学校有没有真正的校企合作企业名单，这个名单上有没有知名或正规的企业。如果全是小作坊或者来路不明的公司，要打问号。

三、怎么系统地提前调查
1. 官方渠道先查一遍

全国技工院校信息查询：人力资源和社会保障部官网可以查到正规技工院校名单。
中职学校可在省教育厅网站查办学资质。
天眼查/企查查上查学校对应的举办主体公司，看有没有法律纠纷、异常经营记录。
学校有没有被教育主管部门处罚过？搜索"[学校名] + 违规 + 处罚"。
2. 实地考察时重点看这些

学生下课后的状态——是疲惫压抑还是正常放松？氛围不会骗人。
问一问在校生："你们学校怎么样？"不用拐弯抹角，直接问。学生一般会说实话。
看看宿舍、食堂、实训车间——这些是最能反映管理水平的地方。
观察老师的状态——老师精神面貌怎么样？愿意不愿意停下来跟你说几句？
3. 多平台交叉验证评价

不要只看一个平台，把小红书、知乎、贴吧、微博都搜一搜。
特别注意时间：如果是大量同时间段（比如招生季集中出现）的正面评价，可能是学校组织的。反倒是零散分布、有好有坏的评价更可信。
负面评价要看得具体——是抱怨管理太严、还是反映实习剥削、还是反映教学质量？性质不一样。
4. 问清楚实习安排

实习单位是哪几家企业？
实习岗位是什么？和所学专业对口吗？
实习期间有没有老师驻场指导？
实习报酬是多少？学校抽不抽成？
如果招生老师对这些问题支支吾吾、避而不答，或者反问"你操心这个干嘛"，这就是很明显的信号。

四、一个实用建议
你不需要把所有学校都查一遍，先把自己候选名单里的学校用上面这些方法过一遍，重点盯住两个核心问题：

这个学校有没有真正做事的能力（看实训设备、校企合作、师资）？
这个学校把你当学生还是当劳动力（看实习安排、招生说辞、管理风格）？
你现在是在哪个阶段？是正在选学校，还是已经有一些候选名单了？如果告诉我你在哪个省份、大概是什么类型的学校（三年制中职、五年一贯制、还是高职），我可以帮你更具体地分析。

```
