## 这是什么？为什么做这个？

在今天的教育叙事里，职校生常常不是被真正理解，而是被提前归类。很多人还没来得及认识一个具体的人，就已经先接受了一套现成判断: 分数低一点，就意味着能力差一点；去了职校，就像是“掉队”了；没走普高这条路，好像人生的上升通道也跟着一起变窄了。围绕职业教育的许多讨论，表面上是在谈学校、谈专业、谈出路，实际上谈的却是等级、体面和谁更值得被看得起。

这种偏见最残酷的地方，不是它说得难听，而是它常常被说得理所当然。它会出现在亲戚饭桌上，也会出现在老师的劝说里，出现在同龄人的玩笑里，出现在那些看似“为你好”的建议里。很多职校生不是在某一个瞬间被打击，而是在一次次被比较、被矮化、被替代发言的过程中，慢慢学会怀疑自己。好像只要进了职业教育，就必须先为自己的存在辩解，再去谈梦想、兴趣、能力和未来。

可真正活在这条路上的人，面对的从来不是一句“读职校也有前途”就能解决的事。有人在中考后被仓促分流，甚至还没来得及理解不同学校、不同专业、不同升学路径之间的差别，就被要求马上做决定；有人明明想继续往上读，却连本省职教高考、对口升学、专升本到底怎么走都找不到清楚、可信、讲人话的信息；有人在顶岗实习里遇到超时劳动、工资被扣、协议不透明，却被默认为“学生嘛，忍一忍”；还有人最难受的根本不是学业，而是每天都要承受“你们这种学校”“你以后也就这样了”这种话，一点一点把自尊磨薄。

更现实的问题是，社会并不真的在认真听职校生说话。关于他们的内容很多，但真正从他们的处境出发的内容很少。官方文件讲制度，却不回答“我现在该怎么办”；媒体报道喜欢讲逆袭故事，却不讲大多数人真实要付出的代价；社交平台上的讨论不是猎奇，就是羞辱，要么把职校生当笑话，要么把他们包装成励志样板。很多 AI 也没有好到哪里去: 它们会用空泛的安慰掩盖现实的难题，会在你最需要一个清楚判断的时候，递给你一套看起来完整、其实并不贴身的标准答案。

`职想`就是在这种背景下做出来的。它不是想替谁做人生决定，也不是想替职业教育“洗白”。它更不想顺着那套默认的等级眼光，再把职校生解释一遍、安慰一遍、规训一遍。它想做的事情更直接一些: 当一个职校生、一个正在分流边缘徘徊的初中生、一个焦虑又犹豫的家长，真的带着问题来问时，能有一个工具不回避现实，不重复偏见，不用空话糊弄人，而是把信息讲清，把路径摊开，把代价说明，把那些混在一起的事实、评价、偏见和羞耻一点点分开。

所以你可以把它理解成一个专门为职校生写的 AI 对话 Skill。但如果说得更准确一点，它其实是在试着回答一个更根本的问题: 当一群人长期被忽视、被误解、被低估时，能不能有一个工具，至少先把他们当成值得被认真对待的人。

## 它能干什么

职想目前能支持六种对话场景，根据你说的第一句话自动判断你在哪种情况里：

| 场景 | 典型问题 |
|---|---|
| 🏫 **择校** | 选什么类型的学校、什么专业、如何比较不同学校 |
| 📚 **升学规划** | 职校后怎么继续升学——中高职衔接、职教高考、专升本等 |
| 🏆 **竞赛** | 职校生可以参加哪些竞赛、值不值得参加、怎么选 |
| ✏️ **学业求助** | 课程、作业、考试、某个知识点的学习困难 |
| 💪 **偏见与社会压力** | 因职校身份被羞辱、轻视、否定，或自我怀疑、内化羞耻 |
| 💼 **就业与实习** | 顶岗实习、校企合作、第一份对口工作、劳动权益 |

## 安装 / 使用方法

### 在WorkBuddy中使用

### 在 Cursor 中使用

1. 将 `zhixiang` 文件夹放入你的 Cursor Workspace 的 `.cursor/skills/` 目录下：
   ```
   your-project/
   └── .cursor/
       └── skills/
           └── zhixiang/      ← 把整个文件夹放这里
               ├── SKILL.md
               ├── modes/
               └── references/
   ```

2. 在对话中直接提问即可，不需要任何额外指令。

### 在 Claude Code 中使用

将 `zhixiang` 文件夹放入：

```bash
~/.claude/skills/zhixiang/
```

然后在对话中直接用就行。

### 启用小红书搜索（可选）

小红书搜索需要本地运行 `xiaohongshu-mcp` 服务：

1. 克隆并启动服务：[github.com/xpzouying/xiaohongshu-mcp](https://github.com/xpzouying/xiaohongshu-mcp)
2. 在对话里说「**去小红书查一下**」或「**小红书上怎么说**」即可触发

没有配置这个服务也完全可以正常使用，只是拿不到小红书独家内容。
